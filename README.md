# DataLakehouse ETL Pipeline on Azure

Cloud-based ETL/Data Lakehouse pipeline built with Azure services for automated data ingestion, transformation, and deployment workflows.

## Overview

This project demonstrates the implementation of a modern Azure-based ETL pipeline using Azure Data Factory and GitHub integration. The repository contains Azure Resource Manager (ARM) templates automatically generated through Azure Data Factory publishing workflows.

The project follows core data engineering principles such as:
- automated data workflows
- cloud-based orchestration
- scalable data storage
- version-controlled infrastructure
- deployment automation
